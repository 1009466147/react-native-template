ReactNative 开箱即用的架构模板  
  
内有一个较为完整的 底部tab首页 以及 登录页  
  
启动   
npm run android  
npm run ios  
  
dev工具  
npm run react-devtools  
  
使用最新的RN版本  
   "@react-navigation/material-top-tabs": "^5.1.15",  
   "react": "16.11.0",  
   "react-native": "0.62.2",
